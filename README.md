# linux相关脚本配置

### vim 环境
 - vim 配置环境是来自 [ma6174](https://github.com/ma6174/vim) 这里的配置已经很完善
 - 在vim目录下有setup.sh 也是从ma6174添加直接执行可以安装vim环境
 - 安装好后可以修改.vimrc配置 如创建文件是插入文件头、快捷键等
